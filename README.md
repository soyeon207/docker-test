## 환경 
front 는 node.js<br>
backend 는 flask <br>
DB 는 mysql 을 사용합니다. 

## 실행 방법 
1. 해당 repository 를 clone 받습니다. 
```
git clone https://github.com/soyeon207/docker-test.git
```

2. docker-compose 를 사용해 docker 를 실행시켜줍니다.
```
docker-compose up 
```

3. URL 로 접속해 정상적으로 실행되는 걸 확인해줍니다. 
```
localhost:8080
```
